## What is Freewrite?

Freewrite is a minimalist writing app designed to get you into a flow state. Set a timer, start typing, and let your thoughts flow without distraction.

**Key features:**
- **Focus Timer** — Set a duration and write without stopping
- **Disable Backspace** — Optional mode to prevent editing while writing
- **AI Integration** — Send your freewrite to ChatGPT or Claude for reflection
- **Local Storage** — Entries saved as `.md` files on your device
- **Dark Mode** — Easy on the eyes, day or night
- **Multiple Fonts** — Choose the typography that works for you

## Development

### Prerequisites

- Node.js 18+
- npm or yarn
- Expo CLI

### Run Locally

```bash
# Install dependencies
npm install

# Start development server
npx expo start

# Run on iOS simulator
npm run ios

# Run on Android emulator
npm run android
```

| Technology | Purpose |
