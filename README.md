## Development

### Run Locally

1.  **Install dependencies**
    ```bash
    npm install
    ```

2.  **Start the app**
    ```bash
    npx expo start
    ```

### Tech Stack

-   **Framework**: React Native (Expo)
-   **Language**: TypeScript
-   **Icons**: Lucide React Native
